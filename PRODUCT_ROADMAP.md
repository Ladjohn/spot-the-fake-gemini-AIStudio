# Spot The Fake (Gemini) – Viral Product Roadmap

## Vision
Turn this project into a viral, shareable AI detection game.

---

## Phase 1: Core Game Loop (Week 1)
- Show user AI vs Real content
- 5–10 second timer per question
- Score tracking
- Final score screen

---

## Phase 2: Viral Mechanics (Week 2)
- Add leaderboard (global)
- Add streak system
- Add shareable score card
- Add "Challenge a Friend" link

---

## Phase 3: Content Engine
- 50+ questions (AI vs real images/text)
- Daily new questions
- Difficulty levels

---

## Phase 4: Monetization
- Ads after every 2–3 rounds
- Pro version (₹99): no ads + hard mode

---

## Phase 5: Growth
- Instagram Reels / YouTube Shorts
- Hook: "Only 1% can pass this test"
- Link to game

---

## Tech Stack
- Frontend: React / Next.js
- Backend: Firebase
- AI: Gemini API

---

## Key Metric
- Shares per user (viral coefficient)
